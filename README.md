## - Navedex-API

```bash
# Clone o repositório
$ git clone https://github.com/stilljag/Navedex-API.git

# Acesse o diretório.
$ Navedex-API/api

# Instale as dependências
$ yarn

# Execute as migrations
$ yarn typeorm migration:run

# Execute o app
$ yarn dev

```
